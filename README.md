# Blinkit Sales Analysis Dashboard

## Project Overview

This project analyzes Blinkit's sales performance using an interactive Excel dashboard.

## Business Requirement

The objective of this project is to analyze sales performance and identify key business insights related to:

* Sales Performance
* Product Categories
* Outlet Performance
* Customer Preferences
* Location Performance

## KPIs

* Total Sales: ₹1.2M
* Average Sales: ₹141
* Number of Items: 8.5K+
* Average Rating: 4.0

## Key Insights

* Fruits and Vegetables generated the highest revenue.
* Outlets established in 2018 recorded the highest sales.
* Low Fat products were preferred over Regular products.
* Tier 3 outlets contributed the highest sales.
* Medium-sized outlets generated the most revenue.
* Supermarket Type 1 was the top-performing outlet type.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

## Conclusion

This dashboard provides actionable insights into Blinkit's sales performance and helps support data-driven business decisions.
